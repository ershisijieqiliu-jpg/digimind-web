# DigiMind 本地购买决策工具

无需填写产品型号：选择用途和预算后，点击“生成我的选购简报”，DigiMind 会直接生成购买方向、加钱/降级方案、避坑提示，并给出京东、淘宝、拼多多和品牌官网的搜索入口。

填写一款或两款候选型号仍然可用，可获得适配性判断或两机对比。

## 长期公网部署（Render）

项目可直接用 Docker 部署到 Render，部署完成后会获得任何人均可访问的 HTTPS 网址。

1. 将本目录推送到一个 GitHub 仓库。
2. 在 Render 创建 **Blueprint**，选择该仓库；平台会读取 `render.yaml`。
3. 在 Render 的环境变量中填写 `DEEPSEEK_API_KEY`，不要把密钥提交到 GitHub。
4. 部署完成后，用 `/health` 验证服务状态。

请在 Render 中自行选择适合的套餐。免费套餐通常会休眠，不适合持续在线；实际套餐价格和政策请以 Render 页面为准。

包含需求挖掘、DeepSeek 问答、候选产品比较，以及京东、淘宝、拼多多和品牌官网搜索入口。

## 启动

1. 复制 `.env.example` 为 `.env`，填写 DeepSeek API Key。
2. 在本目录运行：

```powershell
node server.js
```

3. 打开 http://localhost:8091

产品价格、库存、补贴和商家售后变化快，必须以平台实时页面为准。若要实现跨平台实时比价，需要接入有正式授权的数据或联盟 API；本项目不会用网页抓取伪造实时价格。
